## 📖 Project Overview

The **Christian Hopper Portfolio** is a production-ready, single-page application (SPA) designed to demonstrate high-level frontend architecture. While the Barbershop and Landscaping projects showcase industry-specific solutions, this project demonstrates the **underlying engineering standards** I bring to every client:

- **BEM Methodology:** Strict CSS naming conventions for zero style leaking.
- **Component-Driven UI:** Modular React architecture for easy updates.
- **Performance First:** Lightweight assets and optimized smooth-scroll logic.
- **Brand Consistency:** A unified "Emerald & Dark" theme designed for professional trust.

---

## ✨ Features

### 🎯 Core Engineering

- **Custom Smooth Scroll:** A centralized utility (`scrollToSection.js`) managing all site navigation.
- **Dynamic Navigation:** A sticky header that responds to scroll depth and includes a mobile-optimized menu.
- **BEM Architecture:** Fully modular CSS that ensures the project remains maintainable as it grows.
- **Data-Driven Lists:** Services and Portfolio sections built using reusable arrays and `.map()` logic.

### 💡 User Experience (UX)

- **Interactive Hero:** High-impact call-to-actions (CTAs) for quotes and work samples.
- **Selected Work Gallery:** Browser-mockup styling for project showcases to simulate a premium agency feel.
- **Lead Capture Form:** A clean, accessible contact gateway with custom-styled focus states.
- **Responsive Fluidity:** Seamless transitions between mobile, tablet, and ultra-wide 1440px displays.

### 🎨 Design Identity

- **Theme:** Modern Deep-Dark (#050505) with Emerald Green (#00bc7d) accents.
- **Typography:** Focused on readability and visual hierarchy using balanced text-wrapping.
- **Atmosphere:** Subtle radial gradients (glows) to add depth without affecting performance.
